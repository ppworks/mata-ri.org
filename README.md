# ZooChat

It's a multi room chat system.

## for rvm user

Check ruby version.

```
cat .rvmrc.sample
```

```
cp .rvmrc.sample .rvmrc
```

Customize .rvmrc for your environment.

```
vim .rvmrc
```

cd .. && cd -
